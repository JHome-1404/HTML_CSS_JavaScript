Custom properties (Variables CSS)

Las variables se tiene que definir dentro de un selector, se comienzan con --
Su sintaxis es: --nombre-cualquiera: valor; y atributo: var(--nombre-cualquiera)
La variable en un elemento padre solo aplica a elementos hijos dentro de el
Si quieres que tu variables existan para todos los elementos en el html entonces colocar la variable en :root
La pseudoclase :root hace referencias a la etiqueta html, pero esta posee mayor peso y especificidad
En style se puede aplicar tambien variables CSS

Min() & Max()

Min() Toma los valores minimo y los plazma indefinidamente, los valores mayores van decendiendo
Max() Toma los valores maximos y los plazma indefinidamente, los valores menores van decendiendo
Pueden mezclarse varias unidades de medida para formar un ejemplo de responsive (No Siendo), toma solo valores minimos o maximos para ello
