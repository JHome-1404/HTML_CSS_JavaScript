Fuentes Externas @font-face & font-display

Fuentes para dispositivos// example.ttf
Fuentes para la Web (Web Open Font Format)// example.woff example.woff2

@font face es una etiqueta y font-display es un atributo
@font-face Nos permite cargar y crear nuestra propia fuente de familia externa (descargada) a nuestro proyecto
font-display nos permite modificar como se vera dichas fuentes
El formato para trabajar en fuentes en la web es .woff & .woff2
