Degradados

Lineal

Podemos cambiar el grado con lineal-gradient(0deg), "deg para grados" para cambiar la posicion de inicio del degradado
Se puede ajustar con % la cantidad de espacio degrado que toma un colocar
Si se coloca seguidos no se hace el degradado
Si la sintaxis esta mal escrita, se toma los valores anteriores escritos o no toma ninguno
