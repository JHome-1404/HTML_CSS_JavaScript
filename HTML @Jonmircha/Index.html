Atributos de input y formularios

readonly atributo solo lectura
value atributo para dar valores
disabled atributo de elementos desabilitado
placeholder etiqueta para colocar mensaje dentro de un input - ejem cuando escribes estados
form etiqueta para crear formularios

name atributo de nombre de variable de cada input



Ejemplo de formularios

formularios login

method atributo de metodos de envio de formularios> get valor por defectos "url"
post envia formularios de forma oculta en la url
autocomplete atributo para autocompletar valores de input - ejem "historial de veces escritas"
label etiqueta para hacer etiquetas
"id" y "for" atributos de la mano - for para label y el id para input

Importante

Nota// el for en label conecta la etiqueta con el id que contenga el input...tener siempre el mismo nombre o valor;.....el name en input es para nombre o identicador de la variable en javascript

required atributo para mensajes de requerimientos en input
pattern atributos para patrones - ejem requisistos que siempre piden cuando creas contraseñas
autocomplete se pude en form o en input





Importante

Nota// cualquier tipo de validacion o requerimientos de los input se hacen en javascript
